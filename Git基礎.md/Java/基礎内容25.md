#Javaレッスン25
##繰り返しでHTMLを作成
```
// 年齢入力のプルダウン作成
public class Main {
    public static void main(String[] args) {
        System.out.println("<select name='age'>");
        for(int i = 1; i <= 10; i++) {
            System.out.println("<option>" + i + "歳</option>");
        }
        System.out.println("</select>");
    }
}
```
